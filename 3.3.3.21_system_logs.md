# 3.5.2.21. Bảng dữ liệu: SYSTEM_LOGS

## 1. Mô tả
Bảng lưu trữ log lỗi và thông tin debug của hệ thống (Error Log). Dùng cho đội ngũ phát triển theo dõi sức khỏe hệ thống. (Feature Planned).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | Level | varchar(20) | NOT NULL | Mức độ log (Error, Warning, Info) |
| 3 | Message | text | NOT NULL | Nội dung thông báo lỗi |
| 4 | Exception | text | NULL | Stack trace chi tiết (nếu có Exception) |
| 5 | Source | varchar(100) | NULL | Nguồn phát sinh lỗi (Controller/Service name) |
| 6 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm ghi log |
