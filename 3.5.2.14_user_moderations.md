# 3.5.2.14. Bảng dữ liệu: USER_MODERATIONS

## 1. Mô tả
Bảng lưu trữ lịch sử các án phạt hoặc cảnh cáo mà Admin áp dụng lên người dùng (Ban, Mute, Warn).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | ProfileId | uuid | FK (PROFILES), NOT NULL | ID người bị phạt |
| 3 | AdminId | uuid | FK (PROFILES), NOT NULL | ID của Admin ra quyết định |
| 4 | AdminActionId | uuid | FK (ADMIN_ACTIONS), NOT NULL | ID hành động Audit Log tương ứng |
| 5 | ActionType | integer | NOT NULL | Loại án phạt (Ban, TempBan, Warn...) |
| 6 | Reason | text | NOT NULL | Lý do phạt |
| 7 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm áp dụng |
| 8 | ExpiresAt | timestamp | NULL | Thời điểm hết hạn (nếu phạt có thời hạn) |
| 9 | RevokedAt | timestamp | NULL | Thời điểm hủy án (nếu được tha sớm) |
| 10 | Active | boolean | NOT NULL, DEFAULT true | Còn hiệu lực hay không |
