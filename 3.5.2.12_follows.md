# 3.5.2.12. Bảng dữ liệu: FOLLOWS

## 1. Mô tả
Bảng tự liên kết (Self-referencing) lưu mối quan hệ theo dõi giữa các người dùng.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | FollowerId | uuid | PK, FK (PROFILES), NOT NULL | ID người theo dõi |
| 2 | FolloweeId | uuid | PK, FK (PROFILES), NOT NULL | ID người được theo dõi (KOL) |
| 3 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm bắt đầu theo dõi |
