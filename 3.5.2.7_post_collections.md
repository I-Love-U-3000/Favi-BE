# 3.5.2.7. Bảng dữ liệu: POST_COLLECTIONS

## 1. Mô tả
Bảng trung gian thực hiện mối quan hệ nhiều-nhiều (N-N) giữa `POSTS` và `COLLECTIONS`. Một bài viết có thể nằm trong nhiều bộ sưu tập, và một bộ sưu tập chứa nhiều bài viết.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | PostId | uuid | PK, FK (POSTS), NOT NULL | ID bài viết |
| 2 | CollectionId | uuid | PK, FK (COLLECTIONS), NOT NULL | ID bộ sưu tập |
