# 3.5.2.2. Bảng dữ liệu: PROFILES

## 1. Mô tả
Bảng lưu trữ thông tin hồ sơ chi tiết của người dùng. Có mối quan hệ 1-1 chặt chẽ với bảng USERS. Chứa các thông tin hiển thị công khai và cài đặt riêng tư.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, FK (USERS), NOT NULL | Khóa chính, đồng thời là khóa ngoại tham chiếu đến bảng USERS |
| 2 | Username | varchar(50) | UNIQUE, NOT NULL | Tên người dùng định danh (duy nhất) |
| 3 | DisplayName | varchar(100) | NULL | Tên hiển thị trên giao diện |
| 4 | AvatarUrl | text | NULL | Đường dẫn ảnh đại diện |
| 5 | CoverUrl | text | NULL | Đường dẫn ảnh bìa |
| 6 | Bio | text | NULL | Tiền sử/Mô tả bản thân |
| 7 | Role | integer | NOT NULL, DEFAULT 0 | Redundant field (cache từ User role) |
| 8 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm tạo hồ sơ |
| 9 | LastActiveAt | timestamp | NULL | Thời điểm hoạt động gần nhất |
| 10 | PrivacyLevel | integer | NOT NULL, DEFAULT 0 | Cài đặt quyền riêng tư mặc định của hồ sơ (0: Public, 1: Private) |
| 11 | FollowPrivacyLevel | integer | NOT NULL, DEFAULT 0 | Cài đặt quyền ai có thể Follow |
| 12 | IsBanned | boolean | NOT NULL, DEFAULT false | Trạng thái bị khóa tài khoản |
| 13 | BannedUntil | timestamp | NULL | Thời điểm mở khóa (nếu bị khóa có thời hạn) |
