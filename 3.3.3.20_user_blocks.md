# 3.5.2.20. Bảng dữ liệu: USER_BLOCKS

## 1. Mô tả
Bảng lưu trữ danh sách chặn người dùng (Block). Khi A chặn B, B sẽ không thấy bài viết hay tương tác được với A. (Feature Planned).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | BlockerId | uuid | PK, FK (PROFILES), NOT NULL | ID người thực hiện chặn |
| 2 | BlockedId | uuid | PK, FK (PROFILES), NOT NULL | ID người bị chặn |
| 3 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm chặn |
