# 3.5.2.10. Bảng dữ liệu: REACTIONS

## 1. Mô tả
Bảng lưu trữ các tương tác cảm xúc (Reaction) của người dùng. Thiết kế đa hình (Polymorphic-like) hỗ trợ react cho Bài viết (`POSTS`), Bình luận (`COMMENTS`), hoặc Bộ sưu tập (`COLLECTIONS`).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | ProfileId | uuid | FK (PROFILES), NOT NULL | ID người thực hiện react |
| 3 | Type | integer | NOT NULL | Loại cảm xúc (0: Like, 1: Love, 2: Haha...) |
| 4 | PostId | uuid | FK (POSTS), NULL | Khóa ngoại trỏ đến bài viết (nếu react bài viết) |
| 5 | CommentId | uuid | FK (COMMENTS), NULL | Khóa ngoại trỏ đến bình luận (nếu react bình luận) |
| 6 | CollectionId | uuid | FK (COLLECTIONS), NULL | Khóa ngoại trỏ đến bộ sưu tập (nếu react bộ sưu tập) |
| 7 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm react |
