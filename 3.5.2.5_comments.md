# 3.5.2.5. Bảng dữ liệu: COMMENTS

## 1. Mô tả
Bảng lưu trữ các bình luận của người dùng dưới bài viết. Hỗ trợ cấu trúc cây (Reply) thông qua cột `ParentCommentId`.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | PostId | uuid | FK (POSTS), NOT NULL | ID bài viết chứa bình luận |
| 3 | ProfileId | uuid | FK (PROFILES), NOT NULL | ID người bình luận |
| 4 | Content | text | NOT NULL | Nội dung bình luận |
| 5 | ParentCommentId | uuid | FK (COMMENTS), NULL | ID bình luận cha (nếu là reply). NULL nếu là bình luận gốc |
| 6 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm bình luận |
| 7 | UpdatedAt | timestamp | NULL | Thời điểm chỉnh sửa gần nhất |
