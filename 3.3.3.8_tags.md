# 3.5.2.8. Bảng dữ liệu: TAGS

## 1. Mô tả
Bảng lưu trữ danh sách các thẻ (Hashtag) được sử dụng trong hệ thống. Dùng để phân loại và tìm kiếm bài viết theo chủ đề.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | Name | varchar(255) | UNIQUE, NOT NULL | Nội dung thẻ hashtag (VD: #Travel, #Food) |
