# 3.5.2.19. Bảng dữ liệu: MESSAGES

## 1. Mô tả
Bảng lưu trữ nội dung tin nhắn trong các cuộc hội thoại.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | ConversationId | uuid | FK (CONVERSATIONS), NOT NULL | ID cuộc hội thoại chứa tin nhắn |
| 3 | SenderId | uuid | FK (PROFILES), NOT NULL | ID người gửi |
| 4 | Content | text | NULL | Nội dung tin nhắn văn bản |
| 5 | MediaUrl | text | NULL | Đường dẫn file đính kèm (nếu có) |
| 6 | IsEdited | boolean | NOT NULL, DEFAULT false | Trạng thái đã chỉnh sửa |
| 7 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm gửi |
| 8 | UpdatedAt | timestamp | NULL | Thời điểm chỉnh sửa |
