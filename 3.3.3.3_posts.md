# 3.5.2.3. Bảng dữ liệu: POSTS

## 1. Mô tả
Bảng lưu trữ các bài viết (status, ảnh, chia sẻ) do người dùng đăng tải. Là thực thể trung tâm của mô hình Content Core.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | ProfileId | uuid | FK (PROFILES), NOT NULL | ID người đăng bài |
| 3 | Caption | text | NULL | Nội dung văn bản của bài viết |
| 4 | Privacy | integer | NOT NULL, DEFAULT 0 | Quyền riêng tư bài viết (0: Public, 1: Followers, 2: Private) |
| 5 | LocationName | varchar(255) | NULL | Tên địa điểm check-in |
| 6 | LocationFullAddress | text | NULL | Địa chỉ chi tiết |
| 7 | LocationLatitude | double | NULL | Vĩ độ |
| 8 | LocationLongitude | double | NULL | Kinh độ |
| 9 | DeletedDayExpiredAt | timestamp | NULL | Ngày hết hạn thùng rác (nếu bài bị xóa mềm) |
| 10 | IsArchived | boolean | NOT NULL, DEFAULT false | Trạng thái lưu trữ (ẩn khỏi timeline nhưng không xóa) |
| 11 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm đăng bài |
| 12 | UpdatedAt | timestamp | NOT NULL | Thời điểm cập nhật gần nhất |
