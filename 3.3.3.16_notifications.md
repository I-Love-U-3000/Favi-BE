# 3.5.2.16. Bảng dữ liệu: NOTIFICATIONS

## 1. Mô tả
Bảng lưu trữ các thông báo hệ thống gửi đến người dùng (Like, Comment, Follow mới...).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | Type | integer | NOT NULL | Loại thông báo (Like, Comment, Follow, System...) |
| 3 | RecipientProfileId | uuid | FK (PROFILES), NOT NULL | ID người nhận thông báo |
| 4 | ActorProfileId | uuid | FK (PROFILES), NOT NULL | ID người tạo ra hành động (người like/comment) |
| 5 | TargetPostId | uuid | FK (POSTS), NULL | ID bài viết liên quan (nếu có) |
| 6 | TargetCommentId | uuid | FK (COMMENTS), NULL | ID bình luận liên quan (nếu có) |
| 7 | Message | text | NOT NULL | Nội dung tóm tắt thông báo |
| 8 | IsRead | boolean | NOT NULL, DEFAULT false | Trạng thái đã đọc |
| 9 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm tạo thông báo |
