# 3.5.2.17. Bảng dữ liệu: CONVERSATIONS

## 1. Mô tả
Bảng lưu trữ thông tin chung của một cuộc hội thoại (Chat). Có thể là hội thoại 1-1 (DM) hoặc hội thoại nhóm (Group).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | Type | integer | NOT NULL | Loại hội thoại (0: DM, 1: Group) |
| 3 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm tạo hội thoại |
| 4 | MutedUntil | timestamp | NULL | Thời điểm tắt thông báo đến (nếu user mute) |
| 5 | LastMessageAt | timestamp | NULL | Thời điểm có tin nhắn cuối cùng (dùng để sort inbox) |
