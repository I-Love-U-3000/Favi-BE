# 3.5.2.18. Bảng dữ liệu: USER_CONVERSATIONS

## 1. Mô tả
Bảng trung gian lưu thành viên tham gia hội thoại. Quản lý trạng thái đọc tin nhắn và vai trò trong nhóm (Admin/Member).

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | ConversationId | uuid | PK, FK (CONVERSATIONS), NOT NULL | ID cuộc hội thoại |
| 2 | ProfileId | uuid | PK, FK (PROFILES), NOT NULL | ID người tham gia |
| 3 | Role | varchar(20) | NOT NULL, DEFAULT 'member' | Vai trò (member, admin, owner) |
| 4 | LastReadMessageId | uuid | NULL | ID tin nhắn cuối cùng đã đọc (dùng tính unread count) |
| 5 | JoinedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm tham gia nhóm |
