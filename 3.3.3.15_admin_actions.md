# 3.5.2.15. Bảng dữ liệu: ADMIN_ACTIONS

## 1. Mô tả
Bảng nhật ký hệ thống (Audit Log) ghi lại mọi hành động can thiệp dữ liệu quan trọng của Admin (Xóa bài, Xử lý báo cáo, Ban user...) để phục vụ tra cứu và minh bạch.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | AdminId | uuid | FK (PROFILES), NOT NULL | ID Admin thực hiện |
| 3 | ActionType | integer | NOT NULL | Loại hành động (DeletePost, ReviewReport, BanUser...) |
| 4 | TargetProfileId | uuid | NULL | ID người dùng bị tác động (nếu có) |
| 5 | TargetEntityType | varchar(50) | NULL | Loại đối tượng (Post, Comment) |
| 6 | TargetEntityId | uuid | NULL | ID đối tượng bị tác động |
| 7 | ReportId | uuid | NULL | ID báo cáo liên quan (nếu xử lý từ báo cáo) |
| 8 | Notes | text | NULL | Ghi chú thêm/Lý do xóa |
| 9 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm thực hiện |
