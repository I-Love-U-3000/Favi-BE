# 3.5.2.6. Bảng dữ liệu: COLLECTIONS

## 1. Mô tả
Bảng lưu trữ các bộ sưu tập (Album) do người dùng tạo để gom nhóm các bài viết yêu thích hoặc theo chủ đề riêng.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | ProfileId | uuid | FK (PROFILES), NOT NULL | ID người tạo bộ sưu tập |
| 3 | Title | varchar(255) | NOT NULL | Tên bộ sưu tập |
| 4 | Description | text | NULL | Mô tả ngắn |
| 5 | CoverImageUrl | varchar(max) | NULL | Ảnh bìa đại diện cho bộ sưu tập |
| 6 | CoverImagePublicId| varchar(100)| NULL | Public ID trên Cloudinary (quản lý file) |
| 7 | PrivacyLevel | integer | NOT NULL, DEFAULT 0 | Quyền riêng tư (0: Public, 1: Private) |
| 8 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm tạo |
| 9 | UpdatedAt | timestamp | NOT NULL | Thời điểm cập nhật |
