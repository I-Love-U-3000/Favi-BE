# 3.5.2.1. Bảng dữ liệu: USERS

## 1. Mô tả
Bảng lưu trữ thông tin tài khoản người dùng, được quản lý và đồng bộ từ dịch vụ xác thực bên thứ 3 (Supabase Auth). Chứa các thông tin cơ bản về định danh và quyền truy cập.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính, ID định danh người dùng (khớp với Supabase Auth Uid) |
| 2 | Email | varchar(255) | UNIQUE, NOT NULL | Địa chỉ Email đăng nhập |
| 3 | Role | integer | NOT NULL, DEFAULT 0 | Vai trò người dùng (0: User, 1: Admin) |
| 4 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm tạo tài khoản |
