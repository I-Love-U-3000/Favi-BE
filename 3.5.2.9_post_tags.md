# 3.5.2.9. Bảng dữ liệu: POST_TAGS

## 1. Mô tả
Bảng trung gian thực hiện mối quan hệ nhiều-nhiều (N-N) giữa `POSTS` và `TAGS`. Liên kết bài viết với các thẻ chủ đề.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | PostId | uuid | PK, FK (POSTS), NOT NULL | ID bài viết |
| 2 | TagId | uuid | PK, FK (TAGS), NOT NULL | ID thẻ hashtag |
