# 3.5.2.11. Bảng dữ liệu: SOCIAL_LINKS

## 1. Mô tả
Bảng lưu trữ các liên kết mạng xã hội (Facebook, Instagram, LinkedIn...) của người dùng, hiển thị trên trang cá nhân.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | ProfileId | uuid | FK (PROFILES), NOT NULL | ID người sở hữu liên kết |
| 3 | Kind | integer | NOT NULL | Loại nền tảng (0: Facebook, 1: Instagram, 2: Website...) |
| 4 | Url | text | NOT NULL | Đường dẫn liên kết |
| 5 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm thêm liên kết |
