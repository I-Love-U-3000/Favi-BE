# 3.5.2.4. Bảng dữ liệu: POST_MEDIAS

## 1. Mô tả
Bảng lưu trữ danh sách các tệp đa phương tiện (Hình ảnh, Video) đính kèm trong một bài viết. Một bài viết có thể có nhiều media.

## 2. Chi tiết thuộc tính

| STT | Attribute Name | Data Type | Constraint | Meaning or Note |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Id | uuid | PK, NOT NULL | Khóa chính |
| 2 | PostId | uuid | FK (POSTS), NOT NULL | ID bài viết chứa media (ON DELETE CASCADE) |
| 3 | Url | text | NOT NULL | Đường dẫn URL file (Cloudinary/S3) |
| 4 | MediaType | integer | NOT NULL | Loại file (0: Image, 1: Video) |
| 5 | OrderIndex | integer | NOT NULL, DEFAULT 0 | Thứ tự hiển thị trong danh sách ảnh/video của bài |
| 6 | CreatedAt | timestamp | NOT NULL, DEFAULT NOW() | Thời điểm upload |
